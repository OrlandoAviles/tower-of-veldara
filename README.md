# Tower of Veldara

A browser-based roguelike combat prototype featuring a dynamic **stagger/chain system** inspired by JRPG mechanics.

Play the demo here-->  https://orlandoaviles.netlify.app/veldara/

---------------------------------

## 🎮 Overview

**Tower of Veldara** is a focused gameplay prototype built in JavaScript, designed around fast-paced combat and layered system interactions.

The project serves as both:
- a **playable combat system**
- a foundation for the larger **Veldara universe**

---------------------------------

## ⚔️ Core Features

- **Stagger / Chain System**
  - Build pressure through attacks
  - Manage decay and timing
  - Trigger high-damage stagger states

- **Role-Based Combat (in progress)**
  - Inspired by JRPG paradigms (e.g. Commando / Ravager dynamics)

- **Dynamic Combat Flow**
  - Real-time interactions
  - Visual feedback through UI and effects

- **Modular System Design**
  - Combat logic separated from rendering and UI
  - Structured for future expansion (scenes, modes, etc.)

----------------------------------

## 🧠 Tech Stack

- JavaScript (Vanilla)
- HTML5 Canvas
- CSS

----------------------------------

## 🚧 Current Focus

- Refining combat feel and responsiveness  
- Expanding stagger system depth  
- Transitioning toward a **roguelike tower loop**

----------------------------------

## 🌌 Project Direction

This project is evolving into a prequel experience:

> **Tower of Veldara** — a roguelike journey through a shifting tower, focused on combat mastery and progression.

It will later connect into a larger-scale JRPG project set in the same world.
